是否应该开放给外部调用的接口

接口

java接口是狭义的接口，只是编程中的一个规范，像模式、AOP之流，局限于某个领域

广义的接口是指对外提供交互，如api接口、usb接口、rest接口

api:接口是应用程序和模块之间相互沟通的方式。提供某些功能的应用程序或者模块为了与其它程序相互沟通，对外公开了某些功能的调用方式方法，是应用程序和模块之间相互沟通的方式
